{light_purple}Ключ Контейнера{} может стать весьма полезным инструментом для тех, кто хочет защитить свои вещи. Нажав {gold}Shift+Правый клик{} по любому контейнеру (например, сундуку, печи и т.д.), он будет заблокирован и его можно будет открыть только с помощью этого {gold}единственного{} ключа, пока не решат разблокировать его снова. Сделать это так же просто — {gold}Shift+Правый клик{} с ключом.
